    QCM pour débuter⚓︎

    Je vérifie ma compréhension (bouton en bas pour recommencer)

    1. Soit le script suivant : 
      1

      def fonction(x) :
    if x <= 0 :
        return 0
    else :
        return 1

      L'appel fonction(3) renvoie la valeur :

      • 0

      • 1

      1
2
3
4
5
      def fonction(x) :
    if x <= 2 :
        return 3*x-1
    else :
        return x+3

      L'appel fonction(10)-fonction(0) renvoie la valeur :

      • 30

      • 26

      • 14

      • 10

    if p%2 == 0 :
        return p//2
    else :
        return 3*p+1

      • 6
