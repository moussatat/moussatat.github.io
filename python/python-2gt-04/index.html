    Les commentaires en python⚓︎

    Les commentaires sont des parties du code que l'interpréteur doit ignorer. Ils servent à annoter certaines parties du code.

    exécuter le code suivant, et vérifier qu'aucune erreur ne se produit

    ###(Dés-)Active le code après la ligne # Tests (insensible à la casse)
    (Ctrl+I)
    Entrer ou sortir du mode "deux colonnes"
    (Ctrl+Clic pour inverser les colonnes)
    Entrer ou sortir du mode "plein écran"
    Tronquer ou non le feedback dans les terminaux (sortie standard & stacktrace / relancer le code pour appliquer)
    Si activé, le texte copié dans le terminal est joint sur une seule ligne avant d'être copié dans le presse-papier

    la valeur absolue

    L'instruction abs() renvoie la valeur absolue d'une valeur a définie par

    \[ |a| = \begin{cases} a & \text{si } a\geqslant 0 \\ -a & \text{si } a <0 \end{cases} \]

    Tronquer ou non le feedback dans les terminaux (sortie standard & stacktrace / relancer le code pour appliquer)
    Si activé, le texte copié dans le terminal est joint sur une seule ligne avant d'être copié dans le presse-papier

    Tronquer ou non le feedback dans les terminaux (sortie standard & stacktrace / relancer le code pour appliquer)

    On retiendra que pour tout \(a\in\mathbb{R}\), \(|a|\geqslant 0\).

    Importer des modules en Python⚓︎

    Certaines instructions ne sont pas natives, vous devez les importer directement depuis des librairies python.

    importer une instruction contenue dans la librairie maths

    1. Exécuter le script ci-dessous et vérifier qu'il retourne une erreur de type NameError
    2. Décommenter (enlever le signe # du commenaire), et exécuter le script de nouveaux.
      ###(Dés-)Active le code après la ligne # Tests (insensible à la casse)
      (Ctrl+I)
      Entrer ou sortir du mode "deux colonnes"
      (Ctrl+Clic pour inverser les colonnes)
      Entrer ou sortir du mode "plein écran"
      Tronquer ou non le feedback dans les terminaux (sortie standard & stacktrace / relancer le code pour appliquer)
      Si activé, le texte copié dans le terminal est joint sur une seule ligne avant d'être copié dans le presse-papier
