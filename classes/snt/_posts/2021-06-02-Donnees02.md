## Exercice 4 Lire un fichier csv à l'aide de Python

1. Télechargez &laquo; [fichier1.csv <i class="fas fa-file-csv"></i>](https://drive.google.com/file/d/1ISDhHurlQUjyRKSrEIM5eqKsYCMHYn_T/view) &raquo; et &laquo; [fichier2.csv <i class="fas fa-file-csv"></i>](https://drive.google.com/file/d/1IfeQmxgisjn7rAC5JXJEfsaicjehctZh/view) &raquo; puis téléversez les dans votre interface Python à l'aide du bouton <i class="far fa-folder-open"></i>.
1. Saisir et exécuter le script suivant (copier-coller, pour exécuter dans notebook, il faut taper simultanément sur les touches Shift+Enter) : 
{% highlight python linenos %}
import pandas as pa
donnees = pa.read_csv("fichier2.csv", sep=';')
{% endhighlight %} 
- La ligne 1 invoque le module ```pandas``` sous le **pseudonyme** ```pa```.  
- La ligne 2 crée un tableau, nommé ```T``` , qui contient les données de ```fichier2.csv```.  
- Le paramètre ```sep=';'``` indique à Python que les éléments de ```fichier2.csv``` sont séparés par &laquo; ; &raquo;.
